# 离线二维数点

[离线二维数点](https://www.luogu.com.cn/problem/P10814) 问题给定长为 $n$ 的序列，有 $m$ 次询问，每次询问 $[l,r]$ 区间中小于等于 $x$ 的元素个数。

可以简单的通过树状数组完成，我们先把查询和序列全部读入存入结构体，对于序列中的数字左右边界存为 $-1$。按照双关键字排完序以后遍历数组，如果这是序列中的元素，加入树状数组；如果是查询，则求区间内的数并记录在答案数组。

```cpp
struct node {
    int l, r, x, id;
    bool operator<(const node& item) {
        if (x != item.x) x < item.x;
        return l < item.l;
    }
};
```
