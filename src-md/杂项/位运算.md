# 位运算

计算机中所有数据都以二进制形式存储（$0$ 和 $1$ 组成的序列），位运算就是直接对这些二进制位进行操作，由于跳过了十进制转换，运算速度远快于普通算术运算。

### 1. 按位与（&）

两个二进制位都为 $1$ 时，结果为 $1$；否则为 $0$。

### 2. 按位或（|）

两个二进制位只要有一个为 $1$，结果为 $1$；否则为 $0$。

### 3. 按位异或（^）

两个二进制位不同时为 $1$，相同时为 $0$。

### 4. 左移（<<）

将二进制位整体向左移动 $n$ 位，右侧补$0$。

### 5. 右移（>>）

将二进制位整体向右移动 $n$ 位，左侧补符号位（正数补 $0$，负数补 $1$）。



## 二进制枚举

二进制枚举是位运算最经典的应用之一，核心思想是用一个整数的二进制表示集合的子集一位对应一个元素是否被选中（$1$ 表示选中，$0$ 表示未选中）。

```cpp
for (int mask = 1; mask < (1 << n); mask ++ )
    for (int i = 0; i < n; i ++ )
        if (*mask >> i) & 1))

```
